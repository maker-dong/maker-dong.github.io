# linux免密登录

## 客户端生成公私钥
本地客户端生成公私钥：（一路回车默认即可）
````
ssh-keygen
````

上面这个命令会在用户目录.ssh文件夹下创建公私钥
````
cd ~/.ssh
````

下创建两个密钥：
````
id_rsa （私钥）
id_rsa.pub (公钥)
````

## 上传公钥到服务器
````
ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.235.22
````

上面这条命令是写到服务器上的ssh目录下去了
````
cd ~/.ssh

vim authorized_keys
````
可以看到客户端写入到服务器的 id_rsa.pub （公钥）内容。

## 测试免密登录
客户端通过ssh连接远程服务器，就可以免密登录了。
````
ssh root@192.168.235.22
````
